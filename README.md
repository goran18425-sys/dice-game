# Dice Betting Game (Advanced CLI)

A command line dice betting game written in C with advanced game mechanics, win streak bonuses and session statistics.

The player bets on a number between 1 and 6. A dice is rolled randomly, and rewards depend on accuracy, streak and proximity to the correct number.

## Features

- Random dice rolling (1–6)
- Player balance system with starting input
- Minimum bet requirement ($10)
- Win streak system with bonus multipliers
- Near-miss bonus (±1 during streak)
- Maximum balance tracking
- Session summary and statistics
- Input validation
- Smooth CLI experience with dice roll delay

## Game Rules

- Choose a number between **1 and 6**
- Enter a bet (minimum $10)
- If your number matches the dice:
  - Win **3x your bet**
- If you have a streak of 2+ wins:
  - Win **5x your bet**
- If you're close (±1) during a streak:
  - Get **1.5x bonus**
- Otherwise:
  - Lose your bet

## Game Flow

1. Set your starting balance (max $1000)
2. Choose a number (1–6) or 0 to quit
3. Enter your bet amount
4. Dice rolls with delay
5. Game result is calculated
6. Balance updates
7. Game continues while balance ≥ $10
